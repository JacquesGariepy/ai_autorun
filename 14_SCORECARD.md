# AI Output Scorecard

Score the AI output or batch on 100.

- Real code understanding: /15
- CSV/backlog compliance: /10
- No hallucination: /15
- Code quality: /10
- Tests: /10
- Security: /10
- Privacy: /5
- Maintainability: /10
- Model management compliance: /10
- Immediate usefulness: /5

Verdict:
- 90-100: excellent
- 75-89: usable
- 60-74: partial
- 40-59: dangerous
- 0-39: rejected
